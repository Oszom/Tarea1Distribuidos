Recomendaciones:
    - Al momento de ingresar el tipo de cliente porfavor respetar el formato mostrado en pantalla, respetando
    mayúsculas y minúscula.
    - Al momento de ingresar una orden porfavor respetar el formato mostrado en pantalla cada campo del pedido 
    separado por "," y sin espacios entre medio.
    - El archivo se encuentra en la carpeta Tarea 1/Tarea1Distribuidos, se recomienda hacer git pull antes de ejecutar el codigo
    
Forma de ejecutar el programa

    - Para correr el servidor de Logística usar el siguiente comando: make runLogistica
    - Para correr el servidor de Camiones usar el siguiente comando: make runCamion
    - Para correr el servidor de Clientes usar el siguiente comando: make runCliente
    - Para correr el servidor de Finanzas usar el siguiente comando: make runFinanzas

    En caso de que de error de protoc correr el siguiente comando en terminal: export PATH="$PATH:$(go env GOPATH)/bin"
